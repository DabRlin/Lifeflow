# 项目概述

## 简介

LifeFlow 是一款面向个人用户的效率管理应用，旨在帮助用户更好地管理日常任务、培养良好习惯、记录生活点滴。应用采用现代化的技术栈，提供流畅的用户体验和美观的界面设计。

## 核心功能

### 1. 任务管理 📋

- **任务创建与编辑**: 快速创建任务，支持标题、描述、分类
- **分类管理**: 自定义分类，按类别筛选任务
- **任务完成**: 一键完成任务，带有动画反馈
- **拖拽排序**: 支持拖拽调整任务顺序

### 2. 习惯追踪 🔥

- **习惯打卡**: 每日打卡记录，培养良好习惯
- **连胜统计**: 自动计算当前连胜和最长连胜天数
- **打卡日历**: 热力图展示全年打卡情况
- **提醒功能**: 设置提醒时间，不错过每次打卡

### 3. 生活记录 📝

- **快速记录**: 简洁的输入界面，随时记录想法
- **时间线展示**: 按时间倒序展示，清晰回顾
- **无限滚动**: 流畅加载历史记录
- **编辑删除**: 支持修改和删除已有记录

### 4. 数据统计 📊

- **今日进度**: 环形图展示当日完成情况
- **趋势分析**: 折线图展示近期打卡趋势
- **年度日历**: 热力图展示全年活跃度
- **关键指标**: 总任务数、完成率、连胜天数等

### 5. 通知系统 🔔

- **打卡提醒**: 习惯打卡时间提醒
- **成就通知**: 达成连胜里程碑时通知
- **系统通知**: 重要系统消息推送

## 设计理念

### Material Design 3 (M3)

应用采用 Google Material Design 3 设计语言：

- **紫色主题**: 主色调 `#6750A4`，贯穿全应用
- **圆角设计**: 大圆角卡片和按钮，现代感十足
- **阴影层级**: 多层阴影系统，增强层次感
- **动画过渡**: 流畅的交互动画，提升体验

### 用户体验

- **简洁界面**: 去除冗余元素，专注核心功能
- **快速操作**: 减少点击次数，提高效率
- **即时反馈**: 操作后立即响应，增强确定感
- **离线优先**: 本地数据库，无需网络即可使用
